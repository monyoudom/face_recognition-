## Build Setup

``` bash
cd udayaufr
# run virtual environments
python3 -m venv
# activate
source tutorial-env/bin/activate
# install dependencies
python3 install -r requirments.txt

# serve with hot reload at localhost:8000
python3 manage.py runserver

# websocket
python3 websocket.py
# Frontend 
python3 main.py

```
